# Automated Logic Cascading System

## Why I Built This:
I wanted to explore how formulas create a "chain reaction" in data. This project shows how one input change can automatically update multiple tables - a core concept for error-free automated dashboards.

## What I Did:
- **Master Table Control:** First table serves as primary input. Changing its value updates all multiplication results instantly
- **Absolute Referencing:** Used (=$H$5) to lock specific values, keeping logic consistent even when data moves
- **Sequential Logic Linking:** Third table connects to second using dynamic formula (=H5+1) - Table 3 stays one step ahead of Table 2
- **Cascading Updates:** Table 4 links to Table 3 same way. One change ripples through all four tables instantly

## Tools Used:
Microsoft Excel (Absolute Referencing, Logical Linking, Sequential Formulas, Table Formatting)
