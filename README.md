🔗 Dynamic Data Synchronization System
💡 Why I Built This?
In data management, manual updates can lead to errors. I built this project to learn how to create a "Single Source of Truth." By linking cells, I ensured that changing data in one master table automatically updates all other connected tables, keeping the information consistent and accurate.
✅ What I Did:
Real-time Synchronization: Linked multiple tables using dynamic cell references (=Sheet1!A1 logic).
Centralized Control: Created a Master Input area that controls the values across the entire workbook.
Data Integrity: Reduced manual entry errors by ensuring that data only needs to be updated once to reflect everywhere.
Workflow Efficiency: Designed the system to save time during large-scale data updates.
🛠️ Tools Used:
Microsoft Excel (Dynamic Cell Referencing, Formula Linking, Data Formatting).
